Core Features

Built for academic excellence

Every module purpose-built for the operational realities of Nigerian higher education.

Human Resource Development
Track every professional development activity across the school with structured records and status management.
  • Training & workshop logs
  • Certification tracking
  • Status: Planned / Ongoing / Completed
  • Per-staff development history
Digital Repository
Institutional archive for research outputs, theses, journals, and conference papers with admin approval workflow.
  • Multi-category submissions
  • Admin approval pipeline
  • Keyword search & filters
  • Download tracking
Courseware System
Complete course material lifecycle — upload, organize by course, and enable student access with one-click enrollment.
  • Notes, slides, assignments, videos
  • Per-course organization
  • Student enrollment management
  • Content preview & download
Security & Access Control
Enterprise-grade authentication with role-based permission scoping for every route and data operation.
  • JWT-based authentication
  • 3-tier role system
  • Activity audit logging
  • Session expiration control
Admin Analytics
Real-time dashboard charts showing platform usage, enrollment trends, repository growth, and staff activity.
  • User & enrollment metrics
  • Material upload trends
  • Repository statistics
  • Development completion rates
Announcements
Institutional notice board with targeted broadcasts to specific user roles or the entire SGES community.
  • Role-targeted notices
  • Priority announcements
  • Admin broadcast controls
  • Dashboard integration
